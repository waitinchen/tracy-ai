## 🎯 技術細節

### Three.js 動畫
- **粒子系統**：1500 個彩色粒子
- **光暈效果**：AdditiveBlending 混合模式
- **波紋動畫**：RingGeometry 擴散效果
- **互動追蹤**：Raycaster 檢測滑鼠位置

### CSS 效果
- **backdrop-filter**：毛玻璃模糊
- **rgba 透明度**：半透明背景
- **漸層背景**：linear-gradient
- **動畫過渡**：transition 和 keyframes

## ⚙️ 自訂設定

### 調整動畫透明度
在 `index.html` 中找到：
```css
#canvas-container {
    opacity: 0.6;  /* 調整這個值（0-1）*/
}
```

### 調整動畫速度
在 JavaScript 中找到：
```javascript
controls.autoRotateSpeed = 0.5;  /* 調整這個值 */
```

### 調整粒子數量
```javascript
const particleCount = 1500;  /* 調整這個值 */
```

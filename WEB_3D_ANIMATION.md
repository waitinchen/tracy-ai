## 🚀 快速開始

### 1. 啟動服務器
```bash
py start_web_chat.py
```

### 2. 訪問網址
```
http://localhost:8001
```

### 3. 開始對話
- 在輸入框輸入文字
- 點擊發送或按 Enter
- 觀看背景動畫，聆聽黃蓉語音

## 🎨 視覺效果說明

### 背景動畫
- **自動旋轉**：能量球會自動緩慢旋轉
- **呼吸光暈**：光暈會呼吸般放大縮小
- **粒子漂浮**：雲霧粒子會緩慢移動
- **互動波紋**：滑鼠移動會產生能量波紋

### UI 效果
- **毛玻璃**：對話框使用 backdrop-filter 模糊效果
- **半透明**：背景動畫透過半透明容器可見
- **漸層按鈕**：發送按鈕有漸層和陰影
- **訊息動畫**：新訊息會淡入顯示

## 💡 技術實現

### Three.js 動畫
- 使用 Three.js 0.159.0
- WebGL 渲染器
- 粒子系統（Points）
- 光暈效果（AdditiveBlending）
- 波紋動畫（RingGeometry）

### CSS 效果
- backdrop-filter: blur(10px) - 毛玻璃
- rgba() 透明度 - 半透明背景
- linear-gradient - 漸層效果
- transition 和 keyframes - 動畫

## ⚙️ 自訂設定

### 調整動畫透明度
編輯 `web_static/index.html`：
```css
#canvas-container {
    opacity: 0.6;  /* 0-1，數值越大越明顯 */
}
```

### 調整旋轉速度
```javascript
controls.autoRotateSpeed = 0.5;  /* 調整數值 */
```

### 調整粒子數量
```javascript
const particleCount = 1500;  /* 調整數值（影響性能） */
```
